Project Work di Filippo Cortese | Informatica per le Aziende Digitali (L-31) - UniPegaso

In questo repository ho caricato i file sorgente del sito web sviluppato per il mio caso di studio sull'azienda Fileni S.p.A. Il sito è stato progettato per divulgare i parametri ambientali estratti dal Bilancio di Sostenibilità 2024 e per permettere all'utente di scaricare il documento integrale.

Tutto il codice è stato scritto nativamente in HTML5 e CSS3, sfruttando le proprietà Flexbox per rendere il layout responsivo sui dispositivi mobili, senza appoggiarmi a framework esterni come Bootstrap per mantenere il progetto leggero.

Trattandosi di un sito statico, non c'è bisogno di configurare alcun ambiente di sviluppo o server locale. Per testarne il funzionamento è sufficiente scaricare i file presenti in questa pagina, metterli in un'unica cartella ed eseguire un doppio clic sul file index.html per aprirlo nel browser. Ho incluso il PDF del report e le immagini direttamente nella cartella per garantire che la pagina e la funzione di download funzionino correttamente anche offline.
